# swiftcase.github.io
My pet project for system analysis
Данный документ описывает учебный (pet-project) который автор выбрал для демонстрации всем заинтересованным лицам навыков и умений автора в системной интеграции. Он создавался в трудные времена хаоса на рынке труда конца 2025 г., когда толпы системных аналитиков были выставлены их любимыми компаниями на рынок угасающей экономики и появления беспрепятственного доступа к чуду искусственного интеллекта. 
За основу был взят абзац описания системы трансграничных платежей SWIFT из сайта одного банка и его основе пройдены этапы системного анализа. 
Дабы это было не утомительным и долгим занятием автор все таки иногда использовал доступный ему искусственный интеллект, но использовал свой там где это было нетрудно и быстро, а также для исправления недостатков искусственного. 
Документ создается постепенно, разделы добавляются, расширяются, детализируются и изменяются по мере возможностей, настроения и потребности. 
В первой половине документа раздел первоначально прорабатывается без ИИ, во второй части с ИИ.
